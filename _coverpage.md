# HelloID Vault Management

> HR data anonymization and management tool for HelloID connectors

- Anonymize vault.json data with European names
- Support for SQLite, Turso, and Supabase (PostgreSQL)
- Manage persons, contracts, contacts, and reference data
- Import with person selection and cascade manager support
- PostgreSQL COPY bulk import (14s for 700+ persons)
