# Keyboard Shortcuts

## Person Detail View

| Shortcut | Action |
|----------|--------|
| **Ctrl+1** | Switch to Information tab |
| **Ctrl+2** | Switch to Contracts tab |
| **Ctrl+3** | Switch to Contacts tab |

## Contracts View

| Shortcut | Action |
|----------|--------|
| **Double-click** | Open contract JSON view |
| **Right-click** | Context menu (Copy, View, Edit, Delete) |

## Manager Search Popup

| Shortcut | Action |
|----------|--------|
| **Enter** | Select highlighted result |
| **Double-click** | Select result and close popup |
| **Esc** | Close popup (via click-away) |

## General

| Shortcut | Action |
|----------|--------|
| **Tab** | Navigate between fields |
| **Enter** | Confirm/Save in dialog forms |
| **Esc** | Cancel/Close dialog |
